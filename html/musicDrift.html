Sudoku - is a logic-based, combinatorial number-placement puzzle. In classic Sudoku, the objective is to fill a 9 x 9 grid with digits so that each column, each row, and each of the nine 3 x 3 subgrids that compose the grid (also called "boxes", "blocks", or "regions") contain all of the digits from 1 to 9. The puzzle setter provides a partially completed grid, which for a well-posed puzzle has a single solution

When you click on the "Settings" button, a settings window opens with the ability to enable or disable the timer, highlight repetitions, highlight the same numbers (all functions are initially enabled).

When you click on the "Leaders" button, a window opens with a table of the top 10 players for each difficulty.

When you click on the "Play" button, a window with a choice of difficulty opens, after choosing a difficulty, a window with the game opens. To win, you need to fill the 9 by 9 grid with numbers so that each row, column and 3 by 3 grid contains all the numbers from 1 to 9. To enter values in empty cells, you need to select a cell and enter a number from the keyboard.

If you enter an incorrect number, it will be highlighted in red, if the item "Highlight repetitions" is enabled in the settings, conflicting numbers will be highlighted in a column and/or row and/or grid.

When you click on a cell with a number and the item "Highlight identical numbers" is enabled in the settings, all cells with the corresponding number on the entire playing field will be highlighted.

When all the fields are filled in with the “Timer” item enabled in the settings, a window will appear asking you to save your result.

To save the result, you need to enter a name (without spaces, no longer than 20 characters) and click on the "Save" button, after which the result will be entered in the leaderboard, and the user will be returned to the main menu.

Pressing the "Exit" button will immediately return the user to the main menu.
